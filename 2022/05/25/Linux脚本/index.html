变量

规则
  • 命名中只能使用英文字母，数字和下划线，首个字符不能用数字开头
  • 中间不能又空格，可以使用下划线
  • 先声明后使用

运算符

文件测试运算符
通配符

  • *：代表匹配任意字符

  • ？：一个？代表一个字符

  • ;：连续不同命令的分隔符

  • [[0-9]]: 匹配0-9之间的10个数字

转义字符
  • \ : 使反斜杠后边的一个变量变成一个单纯的字符串
  • ‘’ : 转义其中所以变量保存为单纯的字符串
  • “ “ : 保留属性，不进行转义处理
  • `` ： 把其中命令执行后返回结果
正则表达式（用于处理文件内容，而不是文件）

  • ^

  • $

  • ^$

  • .

  • \ : 转移字符

  • .*

  • a\{3\}

grep

用于搜索，并打印

  • 参数
    • -i : 忽略大小写
    • -v : 取反
    • -n : 列出行号
    • -c ：只输出数量
    • -o ：只输出匹配的内容
